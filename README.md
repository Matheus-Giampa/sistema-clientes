# Sistema de Clientes

## 📝 Descrição do Problema
Este projeto é um sistema simples para cadastro de clientes.  
O objetivo é demonstrar um CRUD básico (Create, Read, Update, Delete) em Java.

## ⚙️ Funcionalidades
- Cadastrar cliente
- Listar clientes
- Atualizar cliente
- Remover cliente

## 🛠 Tecnologias Utilizadas
- Java 17
- Git/GitHub
